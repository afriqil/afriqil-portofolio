## ⭐ Featured Project

### Inside Airbnb NYC Market Analysis & Power BI Dashboard

An end-to-end data analyst portfolio project analyzing **7.8M+ raw Airbnb records** across listings, calendar, and neighbourhood datasets.

The project was built to understand the New York City short-term rental market by analyzing pricing patterns, location performance, room type distribution, host behavior, estimated revenue, occupancy indicators, and dashboard-ready business metrics.

#### Project Highlights

- Analyzed **7.8M+ raw records**
- Built **4 Power BI dashboard pages**
- Created dashboard-ready fact and dimension tables
- Designed a star schema data model
- Developed DAX measures for KPI reporting
- Performed exploratory and statistical analysis
- Delivered business insights and recommendations

#### Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- DAX
- Data Modeling
- Jupyter Notebook

#### Dashboard Pages

The Power BI dashboard includes:

1. **Executive Overview**  
   High-level KPIs, listing distribution, revenue overview, and market summary.

2. **Borough & Neighbourhood Analysis**  
   Location-based performance analysis across boroughs and neighbourhoods.

3. **Pricing & Revenue Analysis**  
   Price distribution, estimated revenue patterns, room type comparison, and capacity-based insights.

4. **Host & Statistical Insights**  
   Host behavior, superhost analysis, review performance, and statistical patterns.

#### Key Business Questions

This project answers questions such as:

- Which boroughs and neighbourhoods dominate the Airbnb market?
- How does pricing vary by room type, location, and capacity?
- Which areas generate stronger estimated revenue?
- How do host characteristics relate to listing performance?
- What patterns can support pricing and market strategy?

## 🚀 How to Run Locally

Clone this repository:

```bash
git clone https://github.com/afriqil/afriqil-portofolio.git
```

Move into the project folder:

```bash
cd afriqil-portofolio
```

Open the project in VS Code:

```bash
code .
```

Run the website locally using one of these methods:

### Option 1: Open Directly

Open `index.html` in your browser.

### Option 2: Use Live Server

If you use VS Code:

1. Install the **Live Server** extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## 🌍 Deployment with GitHub Pages

This website can be deployed for free using GitHub Pages.

### Deployment Steps

1. Push this project to GitHub.
2. Open the repository on GitHub.
3. Go to **Settings**.
4. Open the **Pages** menu.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch:

```text
main
```

7. Select folder:

```text
/ (root)
```

8. Click **Save**.
9. Wait until GitHub Pages finishes publishing the website.

After deployment, the website URL will look like this:

```text
https://afriqil.github.io/afriqil-portofolio/
```
